# Speech to Text Academic Assistant

This project is a **web-based speech-to-text tool** with:

- Live speech recognition
- Editable AI-style summary
- Auto-save (localStorage)
- Theme toggle: Dark, Light, High-Contrast
- Color presets: Blue, Purple, Green
- Academic mode for note-taking
- PDF & DOCX export
